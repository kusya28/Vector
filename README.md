# Проєкт: Vector Operations with JSON

Ця програма на C++ призначена для генерації векторів чисел, виконання математичних операцій (групування, сумування, сортування) та збереження результатів у форматі JSON.

## 🛠 Функціонал
* **Генерація:** Створення вектора дробових чисел.
* **Групування:** Обчислення сум елементів у групах по $n$ елементів.
* **Сортування:** Реалізація алгоритмів сортування за зростанням та спаданням.
* **Аналіз:** Пошук групи з максимальною сумою та підрахунок дублікатів.
* **Експорт:** Збереження звіту у файл `result.json`.

## 🚀 Як запустити
1. Відкрийте файл рішення `NewRepo.sln` у **Visual Studio 2022**.
2. Переконайтеся, що в папці проєкту присутній файл `json.hpp`.
3. Налаштуйте проєкт як **Console Application** (якщо виникає помилка LNK2019).
4. Натисніть `Ctrl + F5` для запуску.

## 📂 Структура файлів
* `main.cpp` — логіка взаємодії з користувачем.
* `Vector.h` / `Vector.cpp` — реалізація математичних функцій.
* `json.hpp` — бібліотека для роботи з JSON.
* `a.txt` / `b.txt` — проміжні дані.
* `result.json` — фінальний звіт.

## ✅ Тестування
Проєкт охоплений Unit-тестами (Native Unit Test Framework). 
Успішно пройдено 7 тестів для основних функцій:
* Перевірка генерації вектора
* Обчислення суми елементів
* Сортування (ASC/DESC)
* Групування та пошук макс. групи
* Обробка дублікатів

## 📦 Залежності
Для роботи з JSON використовується бібліотека [nlohmann/json](https://github.com/nlohmann/json).
